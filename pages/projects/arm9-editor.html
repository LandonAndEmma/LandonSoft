Mario Kart DS ARM9 Editor

A C# program designed to edit values in Mario Kart DS's ARM9.bin code file.

This cross-platform tool allows modders to customize various aspects of Mario Kart DS by directly editing the game's binary code.

Features

  • Edit ARM9.bin values
  • Cross-platform compatibility (PC)
  • User-friendly interface
  • Open-source
